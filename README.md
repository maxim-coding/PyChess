# вҷҹпёҸ PyChess

PyChess is a desktop chess application developed in **Python** using **Tkinter**. It features a custom-built chess engine, AI powered by the Minimax algorithm with alpha-beta pruning, local and online multiplayer, configurable game clocks, and customisable themes.

Developed as an **HSC Software Engineering Major Project**, PyChess demonstrates software engineering principles including object-oriented programming, algorithm design, networking, GUI development, testing, and version control.

- вҷҹпёҸ Complete chess rules implementation
- рҹӨ– AI opponent using Minimax with Alpha-Beta Pruning
- рҹҢҗ Online multiplayer using shareable game codes

- рҹҺЁ Custom board colours and themes

## AI

The computer opponent uses the **Minimax** search algorithm with **Alpha-Beta Pruning** to efficiently evaluate future board positions.

### Difficulty Levels

- Easy
- Medium
- Hard

Each difficulty adjusts the search depth and evaluation behaviour to provide an appropriate challenge.
